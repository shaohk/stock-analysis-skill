【分析维度】

1. **公司主营产品/服务识别**
   - 核心产品分类（按营收占比排序）
   - 产品特征（标准化/差异化/周期品/消费品）
   - 主要客户群体画像
   - 收入结构明细（地区/产品线/渠道）

2. **上游供应链分析**
   - 主要原材料/元器件识别（按成本占比排序）
   - 原材料价格波动历史与趋势
   - 供应商集中度风险（CR3/CR5）
   - 供应商替代难度与话语权
   - 大宗商品价格对成本端的传导机制
   - 进口依赖度（汇率/关税影响）
   - 供应链瓶颈/稀缺资源分析

3. **下游需求端分析**
   - 终端应用领域拆分（按营收占比排序）
   - 各类终端市场的规模与增速
   - 终端客户的采购模式（长单/现货/招标）
   - 下游客户集中度风险（CR3/CR5）
   - 需求价格弹性分析（提价传导能力）
   - 替代品/替代技术威胁

4. **产业链价值分布（微笑曲线）**
   - 公司在微笑曲线的位置（研发/制造/品牌/渠道）
   - 产业链中哪些环节攫取最大利润
   - 公司向曲线两端延伸的可能性
   - 产业链整合趋势（纵向一体化）

5. **竞争格局分析（波特五力）**
   - 现有竞争者：主要竞争对手、市场份额、竞争烈度
   - 新进入者：壁垒高低、潜在威胁
   - 替代品：替代技术/商业模式威胁程度
   - 供应商议价能力：集中度/替代难度/整合意图
   - 客户议价能力：集中度/信息透明度/转换成本

6. **产业链传导机制**
   - 原材料涨价向成本端的传导路径与时滞（1-3个月/3-6个月/半年以上）
   - 终端需求疲软向营收端的传导路径与时滞
   - 产业链各环节的库存周期分析
   - 产业链利润再分配趋势（哪端在攫取利润）

7. **产业链周期定位**
   - 公司所处行业的周期阶段（萌芽/成长/成熟/衰退）
   - 上游原材料的周期阶段（供不应求/平衡/产能过剩）
   - 下游终端的周期阶段
   - 产业链利润在上下游之间的转移规律

8. **地缘产业链重构（如适用）**
   - 中美贸易战对产业链的重构影响
   - 供应链近岸化/友岸化趋势
   - 关键材料的国产替代进度
   - 产业链转移案例（离开中国/进入中国）

【数据获取】

**第一步：以 WebSearch 为主获取行业产业链数据**

产业链分析需要大量行业研究报告和公开资料：
- 搜索 "XXX行业 产业链结构"
- 搜索 "XXX上市公司 供应商 客户"
- 搜索 "XXX原材料 价格走势"
- 搜索 "XXX行业 竞争格局"
- 行业协会报告、上市公司年报中的产业链描述
- 胡润/福布斯/麦肯锡/高盛/中金公司行业研究报告

**第二步：tushare 获取公司财务和主营构成**

```bash
uv run ~/.claude/skills/stock-analysis/scripts/stock_data_demo.py
```

核心接口（tushare）：
- `stock_company`：公司基本信息（了解主营业务/行业地位）
- `fina_mainbz`：主营业务构成（按行业/产品/地区拆分营收占比）
- `income`：利润表（判断上下游结算周期和议价能力）
- `balancesheet`：资产负债表（应收/应付/存货判断供应链地位）
- `fina_indicator`：财务指标（判断盈利能力和资产质量）

**第三步：akshare 补充详细财务指标**

```bash
uv run ~/.claude/skills/stock-analysis/scripts/akshare_data_demo.py
```

核心接口（akshare）：
- `stock_financial_analysis_indicator(symbol, start_year)`：财务指标（几十项，极详细）
- `stock_financial_abstract(symbol)`：财务摘要（多期对比）
- `stock_profit_sheet_by_report_em(symbol)`：利润表
- `stock_balance_sheet_by_report_em(symbol)`：资产负债表
- `stock_cash_flow_sheet_by_report_em(symbol)`：现金流量表

**第四步：行业板块资金流辅助判断**

```bash
uv run ~/.claude/skills/stock-analysis/scripts/akshare_data_demo.py
```

核心接口（akshare）：
- `stock_sector_fund_flow_summary()`：行业板块资金流（判断上下游景气度）
